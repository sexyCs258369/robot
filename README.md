# Telegram 群组记账机器人 🤖💰

一个功能强大的Telegram群组记账机器人，帮助群组成员轻松记录和管理共同支出。

## 主要功能

### 📊 基础记账功能
- **快速记账**：使用简单命令记录支出 `/spent 金额 项目 [备注]`
- **多货币支持**：支持美元(USD)、人民币(CNY)、欧元(EUR)等多种货币
- **智能分类**：自动识别支出类别（棋牌、BC、群贴、餐饮、交通、娱乐等）
- **备注记录**：为每笔支出添加详细说明

### 👥 群组成员管理
- **成员自动识别**：自动记录参与记账的群组成员
- **余额查询**：实时查看个人及他人欠款情况
- **昵称管理**：支持显示用户昵称或用户名

### 💰 债务管理
- **自动计算**：智能计算成员间的债务关系
- **最优结算**：生成最少的转账次数来结清所有债务
- **债务提醒**：定期发送债务汇总提醒

### 📈 统计与分析
- **支出统计**：按时间、类别、成员等多维度统计
- **图表展示**：生成直观的饼图和柱状图
- **导出功能**：支持导出CSV格式的记账数据

### 🔔 提醒与通知
- **定期报告**：可设置每日/每周/每月自动发送支出报告
- **还款提醒**：友好的债务提醒功能
- **活动通知**：实时通知新的记账记录
